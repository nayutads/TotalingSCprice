# TweetSC

[![clasp](https://img.shields.io/badge/built%20with-clasp-4285f4.svg)](https://github.com/google/clasp)

Gmail 領収書からスーパーチャット投入額を集計し Spread Sheet へ記録する。

## 使い方

このプロジェクトを clone した後の話

1. パッケージのインストール

```bash
$ yarn
```

2. GAS にログイン

```bash
$ clasp login
```

3. GAS プロジェクトの作成

```bash
$ clasp create --type standalone
```

4. プッシュ

```bash
$ clasp push
```
